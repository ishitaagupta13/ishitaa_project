# Bitcoin Sentiment vs Trader Performance

## Objective

This project analyzes how market sentiment (Fear/Greed) affects trader performance.

##  Data Used

* Bitcoin Fear & Greed Index
* Historical Trader Data

## What I Did

* Cleaned and merged both datasets using date
* Analyzed:
  * Average Profit (Closed PnL)
  * Win Rate
  * Risk (using trade size in USD)
* Created visualizations (graphs, radar chart, bubble chart)

##  Key Findings

* Traders earn the most during **Extreme Greed**
* Win rate is also highest in **Extreme Greed**
* Traders take highest risk during **Fear**
* Fear leads to riskier and less efficient trading

##  Conclusion

Market sentiment strongly impacts trading behavior.
Greed leads to better performance, while fear leads to higher risk and lower efficiency.
